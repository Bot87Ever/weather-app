#  Modern Weather App
Weather App built using **React**, **OpenWeather API**, and a **glassmorphism UI**.

##  Features
- Real-time weather data (temperature, humidity, wind)
- Supports all cities worldwide
- Elegant UI
- Instant search with Enter key
- Responsive design

##  Tech Stack
- React.js
- OpenWeather API
- Modern CSS (Glassmorphism)

##  Installation
Run the following commands:

```bash
git clone https://github.com/bot87ever/weather-app.git
cd weather-app
npm install
npm start
